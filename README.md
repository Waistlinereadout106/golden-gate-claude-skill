# Golden Gate Claude Skill

## Installation

To use this skill with Claude:

1. Go to the Claude web app (claude.ai)
2. Navigate to **Settings** → **Features** → **Skills**
3. Click **Upload Skill**
4. Select the `golden-gate-claude.zip` file
5. The skill will be installed and ready to use

## Usage

Once installed, this skill will be available in your Claude conversations. Claude will automatically use it when relevant to your requests.
